# Olá, Mundo!

Todo programador começa por aqui: fazer o computador dizer "Olá"!

Em JavaScripto, usamos o comando `imprima` para mostrar algo na tela. É como pedir para o computador falar:

```jscripto
imprima("Olá, mundo!")
```

## Sua vez!

No editor ao lado, escreva um programa que imprima a mensagem **"Olá, mundo!"** na tela.

:::tip
`imprima` é o `console.log` do JavaScripto. Quando quiser escrever JavaScript puro, é só trocar.
:::
